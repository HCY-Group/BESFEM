Detailed Description

Manages electrolyte concentration calculations.

Manages the electrolyte concentration calculations.

This class is derived from Concentrations and includes functionality for:

  • Setting initial values.
  • Time-stepping methods using Crank-Nicolson.

This class extends Concentrations to include specific functionality for:

  • Initializing electrolyte concentrations.
  • Time-stepping using Crank-Nicolson.
  • Applying boundary conditions.
